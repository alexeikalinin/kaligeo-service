## .env.local шаблон

После получения всех ключей — вставить в файл `.env.local` в корне проекта:

```bash
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
GOOGLE_AI_API_KEY=AIza...
PERPLEXITY_API_KEY=pplx-...
DEEPSEEK_API_KEY=sk-...
YANDEXGPT_API_KEY=AQVN...
YANDEX_FOLDER_ID=b1g...
GIGACHAT_CLIENT_ID=...
GIGACHAT_CLIENT_SECRET=...
DATABASE_URL=postgresql://...
DIRECT_URL=postgresql://...
TRIGGER_SECRET_KEY=tr_...
NEXT_PUBLIC_APP_URL=http://localhost:3000
RESEND_API_KEY=re_...
FROM_EMAIL=noreply@kaligeo.com
BLOB_READ_WRITE_TOKEN=vercel_blob_...
UPSTASH_REDIS_REST_URL=https://...
UPSTASH_REDIS_REST_TOKEN=...
ADMIN_SESSION_TOKEN=  ← любой UUID: node -e "console.log(require('crypto').randomUUID())"
INTERNAL_SECRET=      ← любой UUID: node -e "console.log(require('crypto').randomUUID())"
```
