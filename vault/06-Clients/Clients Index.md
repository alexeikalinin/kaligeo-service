## Как использовать

**Добавить аудит после завершения:**
```
/vault-add-audit <jobId>
```

**Проанализировать нишу перед новым аудитом:**
```
/vault-analyze-niche <название ниши>
```

**Получить контекст ниши для action plan:**
```
/vault-niche-context <ниша> <компания>
```
