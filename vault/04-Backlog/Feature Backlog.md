### Google Stitch — AI Website Fix сервис
**Статус:** идея  
**Приоритет:** P1  
**Описание:** Дополнительный платный сервис. После аудита предлагать клиенту AI-переработку сайта через Google Stitch. Данные аудита (слабые места, конкуренты) используются как ТЗ для генерации новых страниц.  
**Монетизация:** $150-300 за "Website AI-Optimization Kit"  
**Детали:** [[Google Stitch Integration]]  
**Как работает:**
- Stitch MCP сервер подключается к Claude Code
- По данным аудита генерируем redesign предложения
- Клиент получает готовый код (Tailwind/HTML)
