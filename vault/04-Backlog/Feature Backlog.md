## Конверсионный флоу — детали

### Рекомендуемый вариант: авто-скан после заявки

**Флоу:**
1. Клиент заполняет форму на лендинге → `POST /api/contact`
2. Сервер сразу запускает `FreemiumScan` по домену из поля "сайт"
3. Через ~10-15 мин клиент получает email: "Ваш AI-отчёт готов" + ссылка на `/preview/{scanId}`
4. Страница preview показывает частичные данные (3-4 платформы, без PDF, без action plan)
5. CTA: "Разблокировать полный отчёт" → страница выбора тарифа → оплата → полный аудит

**Почему это работает лучше чем просто письмо:**
- Клиент видит реальные данные по своему бизнесу — не абстрактное обещание
- Создаёт FOMO: видит что данные есть, но часть скрыта
- Нет ручного шага от администратора для первого касания
- Время до ценности: 15 мин вместо "несколько часов"

**Технически уже есть:**
- `FreemiumScan` модель в БД
- `/api/freemium/scan` — запускает скан
- `/preview/{scanId}` — страница с результатами
- Нужно только: вызвать freemium-скан из `/api/contact` + доработать email с результатами

**Альтернативный вариант (проще):** сразу после заявки отправлять demo-отчёт (`/report/demo`) с именем клиента, без реального скана. Быстрее в реализации, но менее персонализировано.
