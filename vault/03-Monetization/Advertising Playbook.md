| Прогрев | минимум 2 недели перед cold-рассылкой, не более 50 писем/день/ящик первые 30 дней |

```
Письмо 1: Посмотрите — вот что ChatGPT говорит о вашем бизнесе прямо сейчас
Письмо 2: Это не ваша вина — просто никто не объяснял эти правила раньше
Письмо 3: Разрыв растёт каждый день, пока конкурент в AI, а вы нет
Письмо 4: После этого письма данные скана будут удалены
```

### Что нужно реализовать (бэклог)

#### `POST /api/leads/ingest` — единая точка входа лидов

```typescript
// Принимает лид из любого источника
// source: 'apify' | 'vk_lead_form' | 'yandex_landing' | 'telegram_bot' | 'manual'
body: {
  companyName: string
  website?: string
  email?: string
  phone?: string
  niche?: string
  city?: string
  source: LeadSource
}

// Действия:
// 1. Upsert Lead (by email or website)
// 2. Если website → POST /api/freemium/scan (автоматически)
// 3. Trigger.dev → enqueue 'cold-email-sequence' (Instantly.ai API)
// 4. Telegram notify → admin
```

#### Instantly.ai Integration (`lib/instantly.ts`)

```typescript
// Добавить лид в кампанию Instantly
export async function addToInstantlyCampaign(params: {
  email: string
  firstName?: string
  companyName: string
  niche?: string
  city?: string
  campaignId: string
})

// Переменные окружения:
// INSTANTLY_API_KEY=...
// INSTANTLY_CAMPAIGN_ID_COLD=...    // холодная рассылка
// INSTANTLY_CAMPAIGN_ID_FOLLOWUP=... // follow-up после freemium
```

#### n8n Workflow (self-hosted, $0)

```
Trigger: Schedule (каждые 6 часов)
Step 1: Apify → получить результаты последнего run
Step 2: Hunter.io → обогатить email по домену (batch)
Step 3: Deduplicate против таблицы Lead в Postgres
Step 4: Score лиды (формула: рейтинг × отзывы × ниша-приоритет)
Step 5: HTTP POST → /api/leads/ingest (батч)
Step 6: Slack/Telegram → отчёт «добавлено X лидов»
```

### Приоритизация ниш для скрапинга

Скорить каждую нишу по: средний чек × боль AI-невидимости × доступность email

| Ниша | Приоритет | Средний чек | Почему |
|---|---|---|---|
| Стоматологии | ⭐⭐⭐⭐⭐ | 8–15k ₽/пациент | Высокий AI-запрос, легко найти контакты |
| Юридические фирмы | ⭐⭐⭐⭐⭐ | 10–50k ₽/дело | Email на сайте, боль понятна |
| Клиники / медцентры | ⭐⭐⭐⭐ | 5–20k ₽ | Высокий AI-запрос, консервативные владельцы |
| Рестораны (Москва/СПб) | ⭐⭐⭐⭐ | 2–5k ₽/чел | Большой объём базы |
| Агентства недвижимости | ⭐⭐⭐⭐ | высокий | Маркетинг-директора есть |
| Салоны красоты | ⭐⭐⭐ | 3–8k ₽ | Много, но мелкие |
| Фитнес-клубы | ⭐⭐⭐ | 3–10k ₽/мес | Конкуренция высокая |
| Автосервисы | ⭐⭐ | 5–15k ₽ | Email сложнее найти |
