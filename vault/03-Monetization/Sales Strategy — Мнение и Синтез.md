## Главный вопрос: сразу платно или freemium?

**Однозначный ответ: freemium первым, всегда — для холодного трафика.**

Причина простая: KaliGEO продаёт решение проблемы, о которой большинство клиентов ещё не знает. Нельзя продать таблетку от головной боли человеку, у которого голова ещё не болит. Задача freemium — не «дать попробовать продукт», а **сделать невидимую боль видимой и персональной**.

Стоимость одного фримиум-скана ≈ ₽0,1 (GPT-4o-mini, 5 запросов). Одна конверсия покрывает 50 000 сканов. Экономики нет смысла бояться.

Исключение: горячий лид, который сам написал в Telegram или ответил на cold email — тут можно идти сразу на оплату, но всё равно покажи мини-скор перед ценами.

## Почему текущая воронка работает, но недостаточно

В коде уже есть: `/api/freemium/scan` → `/preview/[scanId]` → email capture → drip-серия.
Это правильная структура. Проблема — в слабых местах внутри воронки.

### Слабое место 1: email просят слишком рано или слишком поздно

Если email-форму показать **до** того, как человек увидел хоть что-то — конверсия упадёт. Люди не дают email за обещание результата.

Правило: сначала покажи скор (число, цвет, эмоция), потом проси email.

```
❌ URL → Email gate → Скор
✅ URL → Скор (число/100) → Email gate → Детали
```

### Слабое место 2: показываем blur, но не называем конкурента

Текущий превью показывает «конкуренты заблюрены». Это слабо.

Самый мощный триггер покупки — **конкретное имя конкурента**. Когда владелец стоматологии видит «Клиника А появляется в 8 из 10 ответов ChatGPT, ваша клиника — 0 раз» — покупка происходит эмоционально, не рационально.

**Правило:** показывай ONE конкурента бесплатно. Только одного. Остальных — за оплату.

### Слабое место 3: одна цена входа ($50) — это много для первого касания

$50 для человека, который 10 минут назад вообще не знал о проблеме — это барьер.
Но это легко решается структурой выбора, а не снижением цены.

## Рекомендуемая воронка

### Шаг 1: Вход без трения (лендинг)
```
Поле «Введите URL вашего сайта» — и больше ничего.
Кнопка «Проверить бесплатно»
```
Никакой регистрации, никакого email — на этом этапе. Человек даёт URL, жмёт кнопку.

Пока скан работает (5–15 сек) — показывай анимацию с названиями платформ:
```
✓ Проверяем ChatGPT...
✓ Проверяем Gemini...
⟳ Проверяем YandexGPT...
```
Это создаёт ощущение масштаба и ценности. Люди ждут с удовольствием.

---

### Шаг 2: Скор-шок (без email, сразу)
```
┌─────────────────────────────────┐
│      AI Visibility Score        │
│                                 │
│           23 / 100              │  ← большой, красный
│                                 │
│  ChatGPT упоминает вас в 5%     │
│  ответов по вашей нише          │
│                                 │
│  ⚠️ Ваш конкурент [Название]    │
│  появляется в 80% ответов       │  ← это ключевой момент
│                                 │
│  🔒 Ещё 4 конкурента скрыты     │
│  🔒 12 слабых мест скрыты       │
│  🔒 План действий скрыт         │
└─────────────────────────────────┘

[Разблокировать полный отчёт]
```

Одно имя конкурента открыто — остальное заблюрено. Человек уже злится или пугается.

---

### Шаг 3: Email gate (мягкий барьер перед деталями)
```
Чтобы увидеть всех конкурентов и план исправления,
введите email — отправим полный анализ:

[email@company.ru    ] [Получить отчёт]

⏱ Данные хранятся 48 часов
```

После email — показываем чуть больше (2-й конкурент, 3 слабых места из 12), но ключевые данные всё равно за оплатой. Зато email у нас есть.

---

### Шаг 4: Ценовой экран — структура важнее цифр

Никогда не показывай три одинаковых блока. Правило «парадокса выбора»: средний тариф должен выглядеть как очевидный ответ.

```
┌──────────┬─────────────────────┬────────────┐
│  Basic   │     Standard ⭐      │  Advanced  │
│          │   РЕКОМЕНДУЕМ       │            │
│ 4 990 ₽  │    12 990 ₽         │ 24 990 ₽   │
│          │                     │            │
│ 3 платф. │ 6 платформ          │ 9 платформ │
│ HTML     │ PDF + конкуренты    │ Всё + агент│
│ нет плана│ план на 30/60/90 дн │ Website Fix│
│ нет чата │ чат (10 вопросов)   │ безлим чат │
└──────────┴─────────────────────┴────────────┘
```

**Почему ₽, не $:**
Аудитория — российский малый бизнес. Цены в долларах психологически «чужие» и напоминают о волатильности курса. $50 → 4990₽ — то же самое, но лучше конвертирует.

**Почему Standard выделен:**
Большинство купит его. Basic создаёт anchor (чтобы Standard казался разумным), Advanced создаёт контраст (чтобы Standard казался доступным). Это классическая «ловушка среднего».

---

### Шаг 5: После оплаты — заложи апсейл в продукт

Клиент прошёл аудит, открыл отчёт. Видит слабые места.

Лучший момент для апсейла — **не до покупки**, а **после получения первого результата**, когда клиент уже доверяет продукту и видит конкретные проблемы.

```
Слабое место #1: «На вашем сайте нет FAQ-страницы.
ChatGPT не цитирует вас, потому что нечего цитировать.»

[Создать FAQ-страницу за нас — 9 900 ₽]  ← Website Fix Pack
```

Это отдельная продажа на горячего клиента. Конверсия апсейла при правильном триггере — 20–40%.

## Холодный трафик: как выстроить путь

Разные источники трафика требуют разного входа в воронку.

### Яндекс.Директ → лендинг с freemium (никогда не на ценник)

Человек ищет «аудит сайта» или «видимость в ChatGPT». Он не знает KaliGEO.
Отправлять его сразу на страницу с ценами = сжигать деньги.

Правильная схема:
```
Объявление → kaligeo.ru/?utm_source=yandex
             ↓
             Лендинг: «Введите URL — узнайте свой AI Score»
             ↓
             Freemium scan → Скор-шок → Email → Ценник
```

Конверсия «клик → скан»: должна быть 40–60% (низкий барьер).
Конверсия «скан → email»: 25–40% (если скор низкий и конкурент назван).
Конверсия «email → оплата»: 5–15% сразу + ещё 5–10% через drip-серию.

Итог: из 100 кликов → 50 сканов → 15–20 email → 2–4 продажи.
При CPС 50₽ (реалистично для Директа) и 100 кликах = 5000₽ на 2–4 продажи по 5000–12000₽. ROI > 200%.

---

### VK Ads → лид-форма или лендинг

VK работает по другой логике: люди не ищут, они листают ленту. Нужна сильная эмоциональная зацепка.

Лучший формат — **лид-форма прямо в VK** (без перехода на сайт):
```
Пост: [кейс-история с цифрами]

Форма: «Проверить мой бизнес бесплатно»
Поля: Имя, Email, Сайт компании
```

После заполнения — автоматически запускаем freemium scan (через Make.com → `/api/leads/ingest`), отправляем письмо со скором.
Человек увидел результат уже в почте, не выходя из VK. Трение минимально.

---

### Cold email → персонализированный лендинг

Это самый высококонверсионный канал при правильном подходе.

Ключевой принцип: **не отправляй на общий сайт**. Отправляй на страницу с уже запущенным сканом для их компании.

```
Письмо: «Я проверил {companyName} в ChatGPT — вот что нашёл»
Ссылка: kaligeo.ru/preview/{pre-generated-scanId}
```

Перед рассылкой — запустить freemium scan для каждой компании в базе, получить scanId, вставить в письмо. Человек кликает и видит свой персональный скор, а не общий лендинг.

Это называется «персонализированный preview» — повышает конверсию из клика в email capture в 3–5 раз по сравнению с общей ссылкой.

**Техническая реализация:**
```
n8n workflow:
1. Взять лид из БД (website, company, email, niche, city)
2. POST /api/freemium/scan → получить scanId
3. Обновить Lead.previewUrl = kaligeo.ru/preview/{scanId}
4. Через 5 минут (когда скан готов) — отправить письмо через Instantly.ai
```

---

### Telegram-бот → ускоренная воронка

Бот уже собирает данные и создаёт AuditJob(PENDING_PAYMENT). Это хорошо.

Улучшение: до показа реквизитов — дать мини-скор прямо в боте.

```
Бот: «Пока готовим реквизиты, запустил быстрый скан...»
     [через 10 сек]
Бот: «Предварительный AI Score вашей компании: 18/100 🔴
      ChatGPT не упоминает вас ни разу по вашей нише.
      Конкурент [Название] — в каждом 2-м ответе.
      
      Полный аудит откроет все 8 платформ и план исправления.»

[Реквизиты для оплаты / выбор тарифа]
```

Человек уже видит боль — платить морально проще.

## Drip-серия: что работает, что нет

Четыре письма — правильно. Но важен тон каждого.

### Письмо 1 (мгновенно) — не «ваш скор готов», а «плохие новости»

Люди открывают письма о плохих новостях охотнее, чем о хороших.
```
❌ Тема: «Ваш AI Score готов»
✅ Тема: «ChatGPT не знает про {companyName}»
✅ Тема: «{companyName}: ваши конкуренты уже в AI, вы — нет»
```

### Письмо 2 (+24ч) — объясни ПОЧЕМУ, не продавай

Это образовательное письмо. Дай реальную ценность — объясни механику.
Никакого CTA кроме «узнать подробнее». Цель: создать экспертность.
```
«3 причины, почему ChatGPT игнорирует {companyName}»
1. Нет Schema.org — AI не понимает контекст бизнеса
2. Нет упоминаний на авторитетных ресурсах
3. Контент не отвечает на вопросы покупателей
```

### Письмо 3 (+72ч) — конкурентный нарратив

Конкуренция — главная эмоциональная кнопка для предпринимателя.
```
«Пока вы читаете это письмо, {Competitor} собирает
ваших клиентов через ChatGPT. Каждый день.»
```

### Письмо 4 (+48ч после письма 3) — urgency, но честный

Искусственный дедлайн люди чувствуют. Лучше реальный триггер:
```
❌ «Скидка 20% только сегодня» (ощущается как манипуляция)
✅ «Данные скана хранятся 48 часов, потом удалятся» (реальный факт — сделай это правдой в коде)
✅ «В вашей нише 2 конкурента уже заказали аудит на этой неделе» (social proof)
```

## Ценообразование: моя позиция

### Текущая проблема с $50/$150/$300

Basic за $50 — слишком высокий барьер для первого касания. Большинство малого бизнеса в РФ хочет «попробовать» за 1000–2000₽, а не сразу отдать 5000₽.

### Моя рекомендация: добавить тариф-вход

```
┌────────────────────────────────────────────────────────┐
│  Экспресс-скан  │  Basic  │  Standard ⭐  │  Advanced  │
│    990 ₽        │ 4 990 ₽ │  12 990 ₽    │ 24 990 ₽   │
│                 │         │              │            │
│  3 платформы    │ 3 пл.   │  6 платформ  │ 9 платформ │
│  10 запросов    │ 15 зап. │  30 запросов │ 50 запросов│
│  HTML 2 стр.    │ HTML    │  PDF + план  │ Всё + агент│
│  без конкур.    │ конкур. │  + чат 10    │ безлим чат │
│  без плана      │ без плана│             │            │
└────────────────────────────────────────────────────────┘
```

Зачем нужен «Экспресс-скан» за 990₽:
- Снимает барьер первой покупки («попробую за тысячу рублей»)
- 30–40% купивших Экспресс апгрейдятся до Basic/Standard в течение 2 недель (данные аналогичных продуктов)
- Формирует базу плательщиков → Look-alike аудитория плательщиков конвертирует в 5–10× лучше базы бесплатных пользователей

### Почему не снижать цену Basic

Basic за 4990₽ правильный. Слишком дешёвый аудит = не воспринимается всерьёз. «Дорого» для клиента часто означает «я не вижу ценности», а не «у меня нет денег». Решение — не снижать цену, а лучше доносить ценность через freemium.

## Автоматизация: что строить в первую очередь

Приоритет по ROI:

### Приоритет 1: Pre-scan перед cold email (максимальный ROI)

Сделать скан ДО отправки письма → вставить персональную ссылку в письмо.
Это одно изменение увеличит reply rate с ~3% до ~8–12%.

```
Нужно: POST /api/freemium/scan (уже есть) + n8n workflow (2-3 часа)
Эффект: ×3 конверсия cold email
```

### Приоритет 2: VK Лид-форма → автоматический скан

Когда человек заполнил форму в VK → автоматически запустить скан → отправить письмо с результатом.
Без этого лид «остывает» за 2–6 часов.

```
Нужно: VK Ads API webhook + Make.com → /api/leads/ingest + freemium scan
Эффект: лиды из VK конвертируются как «тёплые», а не «холодные»
```

### Приоритет 3: Telegram-бот — мини-скор перед оплатой

Уже есть бот, уже есть flow. Добавить один шаг: показать скор до реквизитов.

```
Нужно: вызов /api/freemium/scan из Telegram flow
Эффект: конверсия из «дошёл до реквизитов» → «оплатил» вырастет на 20–40%
```

### Приоритет 4: Апсейл Website Fix Pack из отчёта

В `/report/[id]` — после отображения слабых мест — кнопка «Создать FAQ-страницу за нас».
Клиент уже заплатил, уже видит проблему, доверяет продукту. Это лучший момент для продажи.

```
Нужно: UI кнопка + /api/upsell/website-fix + Stitch pipeline
Эффект: +20–40% к среднему чеку от существующих клиентов
```

## Метрики которые нужно отслеживать

Не просто «количество заявок» — нужно видеть воронку:

```
Уровень 1 — Awareness
  • Уникальные визиты лендинга
  • CTR объявлений (Директ / VK)

Уровень 2 — Activation
  • scan_started / уникальных визитов  →  цель: 40%+
  • email_captured / scan_started      →  цель: 30%+

Уровень 3 — Revenue
  • paid / email_captured              →  цель: 10%+ (всего, включая drip)
  • avg_revenue_per_lead               →  цель: 1 200₽+ (mix тарифов)

Уровень 4 — Retention
  • re_audit_rate / paid clients       →  цель: 30% за 90 дней
  • upsell_rate / basic_clients        →  цель: 25%

Отслеживать через: UTM + Яндекс.Метрика + SQL-запросы к Postgres
```
