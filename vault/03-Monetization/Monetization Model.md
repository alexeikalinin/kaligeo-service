## Тарифная сетка (текущая на лендинге)

| Тариф | Цена | Запросов | Платформ | Отчёт |
|---|---|---|---|---|
| Basic | $50 | 15 | 3 | 6 стр, только HTML |
| Standard | $150 | 50 | 6 | 28 стр, PDF + видео 15 мин |
| Advanced | $300+ | 150+ | Все 8 | Мультиязычный, 90d план |

## Расширенная модель (предложение)

### BASIC — $50 «Проверить»
**Для кого:** малый бизнес, первый контакт с AI-аудитом  
**Что входит:**
- 15 запросов × 3 платформы (ChatGPT, Gemini, YandexGPT)
- Базовый HTML-отчёт (без PDF)
- Общий score + топ-3 слабых места
- Email с результатом

**Ограничения:**
- ❌ Нет PDF
- ❌ Нет чата после аудита
- ❌ Нет матрицы конкурентов (только упоминания)
- ❌ Нет action plan от Claude

---

### STANDARD — $150 «Понять»
**Для кого:** средний бизнес, хочет полную картину  
**Что входит:**
- 50 запросов × 6 платформ (+ Claude, Perplexity, DeepSeek)
- PDF-отчёт 28 страниц
- Полная матрица конкурентов
- 30/60/90-дневный план от Claude
- [[Post-Audit Chat]] — **10 вопросов к AI после аудита**
- Интерактивный дашборд

**Ограничения:**
- ❌ Нет Alisa, GigaChat (только при Advanced)
- ❌ Чат ограничен 10 сообщениями
- ❌ Нет регенерации секций отчёта
- ❌ Нет content agent (рекомендации по контенту)

---

### ADVANCED — $300+ «Действовать»
**Для кого:** серьёзный бизнес, хочет инструмент роста  
**Что входит:**
- 150+ запросов × все 8 платформ (включая Alisa, GigaChat)
- Мультиязычный анализ (RU + EN)
- Полный [[Agent System|агентный слой]]:
  - **Безлимитный post-audit чат**
  - **Analysis Agent** — углублённый анализ конкурентов/gaps
  - **Content Agent** — контент-брифы, FAQ, Schema-рекомендации
  - **Report Agent** — регенерация любой секции
- Ежемесячный re-audit по сниженной цене ($150)
- Приоритетная обработка (< 24ч вместо 48ч)

## Google Stitch как генератор монетизации

Google Stitch MCP открывает новые продуктовые возможности поверх существующих тарифов.

### Website Fix Pack $149 (апсейл) — основной use-case Stitch

Stitch генерирует готовый код страниц для клиента на основе данных аудита:

```
Данные аудита (слабые места, конкуренты, ниша, запросы)
    ↓
website-fix-agent.ts → промпт для Stitch
    ↓
Stitch MCP: generate_screen (FAQ, About, Services, Schema.org)
    ↓
ZIP-архив: готовые HTML/Tailwind страницы + инструкция внедрения
    ↓
Клиент получает конкретный код, а не абстрактные рекомендации
```

**Почему работает:**
- Не правим чужой сайт (нет ответственности)
- Клиент получает осязаемый результат
- Stitch = профессиональный дизайн без UX-дизайнера
- Апсейл в конце аудита (когда боль видна) → высокая конверсия

### Нишевые лендинги (лидогенерация через Stitch)

Stitch генерирует персонализированные лендинги для каждой ниши:

| Параметр | Значение |
|---|---|
| Входные данные | ниша + город + vault-кейсы |
| Выход | `/landing/[niche]/[city]` |
| Примеры | `/landing/dentistry/moscow`, `/landing/restaurant/spb` |
| Эффект | Конверсия в 2–3× выше общего лендинга |
| Инструмент | `mcp__stitch__generate_screen_from_text` |

Алгоритм генерации нишевого лендинга:
1. Запрос к vault: кейсы + статистика по нише
2. Промпт → Stitch: заголовок, боли, кейс, CTA — специфичны для ниши
3. Deployed как статичная страница Next.js (`/landing/[niche]/[city]`)
4. UTM-трекинг для отслеживания конверсии

| Мини-аудит | Бесплатно / $19 | — |

| Advanced | $300 | Stitch в Content Agent (FAQ-страница в брифе) |

### Landing Pack $99 — новый апсейл (идея)

После аудита клиент видит: «У вас нет страницы FAQ → ChatGPT вас не цитирует».
Мы предлагаем: сгенерировать FAQ-лендинг через Stitch за $99 (Stitch делает за 5 минут).

**Воронка апсейла Landing Pack:**
```
Отчёт → Action Plan → «Создайте FAQ-страницу» (weak point #1)
    ↓
CTA: «Хотим сделать за вас — $99, готово за 24ч» 
    ↓
Stitch генерирует → ZIP → клиент внедряет
```
