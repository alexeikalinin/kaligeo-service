# Monetization Model

tags: #monetization #tiers #product

## Тарифная сетка (текущая на лендинге)

| Тариф | Цена | Запросов | Платформ | Отчёт |
|---|---|---|---|---|
| Basic | $50 | 15 | 3 | 6 стр, только HTML |
| Standard | $150 | 50 | 6 | 28 стр, PDF + видео 15 мин |
| Advanced | $300+ | 150+ | Все 8 | Мультиязычный, 90d план |

## Расширенная модель (предложение)

### BASIC — $50 «Проверить»
**Для кого:** малый бизнес, первый контакт с AI-аудитом  
**Что входит:**
- 15 запросов × 3 платформы (ChatGPT, Gemini, YandexGPT)
- Базовый HTML-отчёт (без PDF)
- Общий score + топ-3 слабых места
- Email с результатом

**Ограничения:**
- ❌ Нет PDF
- ❌ Нет чата после аудита
- ❌ Нет матрицы конкурентов (только упоминания)
- ❌ Нет action plan от Claude

---

### STANDARD — $150 «Понять»
**Для кого:** средний бизнес, хочет полную картину  
**Что входит:**
- 50 запросов × 6 платформ (+ Claude, Perplexity, DeepSeek)
- PDF-отчёт 28 страниц
- Полная матрица конкурентов
- 30/60/90-дневный план от Claude
- [[Post-Audit Chat]] — **10 вопросов к AI после аудита**
- Интерактивный дашборд

**Ограничения:**
- ❌ Нет Alisa, GigaChat (только при Advanced)
- ❌ Чат ограничен 10 сообщениями
- ❌ Нет регенерации секций отчёта
- ❌ Нет content agent (рекомендации по контенту)

---

### ADVANCED — $300+ «Действовать»
**Для кого:** серьёзный бизнес, хочет инструмент роста  
**Что входит:**
- 150+ запросов × все 8 платформ (включая Alisa, GigaChat)
- Мультиязычный анализ (RU + EN)
- Полный [[Agent System|агентный слой]]:
  - **Безлимитный post-audit чат**
  - **Analysis Agent** — углублённый анализ конкурентов/gaps
  - **Content Agent** — контент-брифы, FAQ, Schema-рекомендации
  - **Report Agent** — регенерация любой секции
- Ежемесячный re-audit по сниженной цене ($150)
- Приоритетная обработка (< 24ч вместо 48ч)

---

## Монетизация через апсейл в продукте

### Триггеры апсейла (где показывать)

```
/report/{id} (Basic/Standard пользователь)
└── Увидел score < 30 → "Хотите узнать как это исправить?" → Upgrade to Advanced
└── Прочитал weak points → "Получите контент-план" → Upgrade
└── Исчерпал лимит чата → "Продолжить диалог" → Upgrade

/chat (новый пользователь)
└── "У меня бюджет ограничен" → предложить Basic
└── "Хочу сравнить с конкурентами" → Standard minimum
```

### Таблица feature gates

| Фича | Basic | Standard | Advanced |
|---|---|---|---|
| Отчёт (HTML) | ✅ | ✅ | ✅ |
| PDF | ❌ | ✅ | ✅ |
| Матрица конкурентов | частично | ✅ | ✅ |
| Action Plan (Claude) | ❌ | ✅ | ✅ |
| Post-Audit Chat | ❌ | 10 сообщ | Безлимит |
| Content Agent | ❌ | ❌ | ✅ |
| Analysis Agent | ❌ | ❌ | ✅ |
| Регенерация секций | ❌ | ❌ | ✅ |
| Alisa + GigaChat | ❌ | ❌ | ✅ |
| Мультиязычность | ❌ | ❌ | ✅ |
| Re-audit (мес.) | ❌ | $100 | $150 |
| Скорость | 48ч | 48ч | <24ч |

```ts
export const TIER_FEATURES = {

  STANDARD: { platforms: 6, queries: 50, pdf: true, chat: true, chatLimit: 10 },
  ADVANCED: { platforms: 8, queries: 150, pdf: true, chat: true, chatLimit: Infinity },
}
```
