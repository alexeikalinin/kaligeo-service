# Lead Generation Strategy

tags: #leadgen #growth #marketing #acquisition

## Цель

**3–10 заявок в день** (90–150/мес) при бюджете **150–400 USD/мес + время**.
Основной продукт на входе: мини-аудит (бесплатно или $19) → конверсия в Basic/Standard/Advanced.

---

## Подготовка (обязательно до запуска каналов)

### Мини-аудит (ключевой инструмент входа)
- **Бесплатно**: 5 запросов × 1 платформа (ChatGPT), score + 1 конкурент
- **$19 «Быстрый старт»**: 10 запросов × 3 платформы, PDF 3 стр, 2 слабых места
- Реализация: `/api/freemium/scan` → `/preview/[scanId]` (уже в коде)
- Связь: [[Freemium Strategy]]

### Лендинг-усилители
- Кейс: «Юрфирма СПб — 0% упоминаний в AI → 34% за 60 дней»
- Кейс: «Стоматология Москва — конкуренты в 7 из 10 ответов AI, клиент — ни разу»
- Счётчик: «Сегодня проверено X компаний» (real-time из БД)
- Квиз-заголовок: **«Проверьте невидимость вашего бизнеса в AI за 2 минуты»**
- Геотаргет: Москва, СПб, Екатеринбург, Новосибирск + СНГ (Казахстан, Беларусь, ОАЭ-диаспора)

## Каналы лидогенерации

### A. Скрапинг + холодная рассылка (основной, низкий бюджет)

**Сбор базы:**
- **Apify** → Google Maps Scraper + Yandex Maps Scraper
  - Фильтры: наличие сайта, рейтинг > 4.0, кол-во отзывов > 30
  - Ниши: рестораны, кафе, салоны красоты, стоматологии, юристы, бухгалтерия, автосервисы, клиники, фитнес, недвижимость
  - Данные: название, сайт, адрес, город, категория, телефон, Instagram/VK
- **Outscraper** — запасной вариант для Яндекс

**Обогащение email:**
- **Hunter.io** / **Snov.io** / **GetProspect** — email по домену
- Дополнительно: парсинг mailto: со страниц сайтов (простой Python-скрипт)
- Целевой формат контакта: owner/director/marketing

**Email-рассылка:**
- **Instantly.ai** (рекомендуется): $37/мес, warmup включён, до 50k писем/мес
- **Smartlead** (альтернатива): $39/мес, лучше для агентств
- Разогрев ящиков: 2–3 недели перед запуском
- Лимит: 30–50 писем/день с одного домена в первый месяц

**Шаблон холодного письма:**
```
Тема: ChatGPT не знает про [Название компании]?

[Имя], я проверил — когда люди спрашивают ChatGPT про [ниша] в [город],
ваш бизнес не упоминается ни разу.

Ваши конкуренты [Конкурент1] и [Конкурент2] появляются в 60%+ ответов.

Это называется AI-невидимость — новая проблема, которую не видит ни Яндекс.Метрика,
ни Google Analytics.

За 48 часов делаю полный аудит — какие AI-платформы вас игнорируют и почему.
Стоимость: $50.

Хотите бесплатный мини-скан прямо сейчас? → [ссылка на freemium]

[Подпись]
```

**Автоматизация воронки:**
```
Apify Scraper
    ↓
Hunter.io / Snov.io (email enrichment)
    ↓
Google Sheets / Airtable (CRM)
    ↓
Instantly.ai (sequence: 3 письма за 7 дней)
    ↓
Ответил → ручная квалификация → Checkout link
```
- **Make.com** (5 операций/мес бесплатно) или **n8n** self-hosted ($0) для автоматизации
- Хранение базы: **Supabase** (уже есть) таблица `LeadProspect`

**KPI скрапинга:**
- 500 контактов/день = реально с Apify
- 15–20% валидных email (после обогащения)
- Response rate холодной почты: 2–5%
- Конверсия ответ → продажа: 10–20%
- **Итого: 2000 контактов/нед → 4–20 продаж/нед**

### B. Telegram (быстрый старт, бесплатно)

**Где искать:**
- Чаты предпринимателей: @biznes_ru, @online_biz, @msk_business
- Чаты маркетологов: @targetolog, @seo_chat_ru
- Чаты конкретных ниш: стоматологи, юристы, рестораторы
- Региональные бизнес-чаты Москвы, СПб, Екб

**Подход:**
1. Нативный пост от имени фаундера: «Провёл эксперимент — проверил 50 компаний в ChatGPT. 80% невидимы. Вот что обнаружил...»
2. Ответы в тредах где обсуждают AI/маркетинг — без спама, с ценностью
3. Личные сообщения активным участникам (до 20/день, иначе бан)

**Telegram-канал KaliGEO:**
- Название: «AI-видимость бизнеса | KaliGEO»
- Контент: кейсы аудитов (с разрешения), статистика по нишам, полезные посты
- Цель: 500+ подписчиков за 3 месяца → тёплая аудитория

### C. LinkedIn (для B2B сегмента, ОАЭ/Европа/СНГ)

**Таргет:** маркетинг-директора, владельцы агентств, head of growth
**Метод:**
- Sales Navigator поиск по должности + отрасль + страна
- Connection request + follow-up через 3 дня
- Шаблон: аналогично холодному email, но короче (3–4 предложения)
- Лимит: 20–30 invite/день (без Premium — 80/мес)

### D. SEO + контент (долгосрочный канал, 3–6 мес)

**Ключевые запросы:**
- «видимость в ChatGPT», «аудит AI», «GEO оптимизация», «как попасть в ответы ChatGPT»
- «что такое AI visibility», «проверить упоминания в AI»

**Контент-план:**
1. «Почему ваш бизнес невидим в ChatGPT: 5 причин» (Habr / VC.ru)
2. «Мы проверили 100 московских ресторанов в 7 AI. Результаты шокируют» (VC.ru)
3. «GEO vs SEO: чем отличается оптимизация для AI» (блог kaligeo.ru)
4. Кейс-стади с цифрами (публикация на Medium + VC.ru)

**Распространение:**
- VC.ru, Habr, Spark.ru — бесплатные публикации с реальной аудиторией
- Дублировать в Telegram-канал + LinkedIn

### E. Партнёрский канал (мультипликатор, нулевой бюджет)

**Целевые партнёры:**
- SEO-агентства (у них клиенты уже платят за видимость)
- SMM-агентства (дополнительный продукт для клиентов)
- Веб-студии (апсейл после разработки сайта)
- Бизнес-клубы и нетворкинг-сообщества

**Условия партнёрства:**
- Реферальная ссылка → 20% от первой продажи
- White-label: партнёр продаёт под своим брендом, мы выполняем
- Реализация: партнёрская таблица в Airtable, UTM-трекинг

### F. Таргетированная реклама (при бюджете 100–200 USD/мес)

**ВКонтакте:**
- Таргет: владельцы бизнеса, маркетологи
- Формат: промо-пост с кейсом + CTA на freemium
- Бюджет: $50–100/мес, начать с $5–10/день

**Яндекс.Директ:**
- Ключи: «аудит бизнеса», «AI маркетинг», «видимость в ChatGPT»
- Лендинг: отдельный под каждый ключ (A/B)
- Бюджет: $50–100/мес

**Ретаргетинг:**
- Пиксель на `/preview/[scanId]` (посмотрели, не купили)
- Аудитория: были на сайте > 60 сек, не конвертировались
- Объявление: «Ваш AI-score всё ещё [X]/100. Как исправить?»

## Google Stitch в воронке лидогенерации

### Персонализированные лендинги по нише (ключевой инструмент)
Stitch генерирует кастомный лендинг для каждой ниши:
- Входные данные: ниша + город + аудит из vault
- Выход: персонализированная страница `/landing/[niche]/[city]`
- Пример: `/landing/stomatology/moscow` — заголовок, кейсы, боли — специфичны для ниши
- **Эффект**: конверсия нишевых лендингов в 2–3× выше общего

### Динамические email-шаблоны
- Stitch генерирует HTML email с результатами freemium-скана
- Персонализация: название компании, score, топ-конкурент, ниша
- Отправка через Resend API (уже запланирован)

### Кейс-стади страницы
- Stitch генерирует красивые PDF/HTML кейс-стади для партнёров
- Автоматически добавляет данные из vault-аудитов

### План запуска

**Неделя 1: Инфраструктура**
- [ ] Настроить Apify → первые 2000 контактов (рестораны Москва + СПб)
- [ ] Обогатить email через Hunter.io
- [ ] Запустить прогрев 2–3 email-ящиков в Instantly
- [ ] Улучшить лендинг: добавить кейс + счётчик аудитов
- [ ] Создать Telegram-канал KaliGEO

**Неделя 2: Первые касания**
- [ ] Запустить холодную рассылку (30/день с каждого ящика)
- [ ] Нативный пост в 3–5 Telegram-чатах предпринимателей
- [ ] Первый пост на VC.ru (эксперимент с 50 компаниями)
- [ ] Настроить Make.com воронку (Apify → Sheets → Instantly)

**Неделя 3: Масштаб**

- [ ] Запустить таргет ВК ($5/день тест)
- [ ] Outreach к 5–10 SEO-агентствам (партнёрство)
- [ ] LinkedIn outreach (если B2B-сегмент)

**Неделя 4: Оптимизация**
- [ ] A/B тест тем писем (3 варианта)
- [ ] Анализ: какие ниши конвертируют лучше
- [ ] Реферальная программа для первых клиентов
