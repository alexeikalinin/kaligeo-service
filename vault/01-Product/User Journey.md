# User Journey

tags: #product #ux #flow

## Полный флоу клиента

```
Лендинг (kaligeo.vercel.app)
    ↓ "Получить аудит"
Веб-приложение /chat
    ↓ AI-ассистент задаёт вопросы
Сбор данных (3-4 сообщения)
    ↓ submit_audit tool
Страница статуса /audit/{id}
    ↓ pipeline ~30-60 мин
Email + /report/{id}?token=...
    ↓
Интерактивный отчёт
    ↓ (Standard/Advanced)
Post-Audit Chat
    ↓ (Advanced)
Ежемесячный re-audit
```

## Детальный сценарий (Standard, $150)

### 1. Лендинг → Чат (0 мин)
- Клиент нажимает "Получить аудит" на лендинге
- Переходит на `app.kaligeo.com/chat` (или `/chat`)
- Видит первый вопрос: "Какую компанию хотите проверить?"

### 2. Диалог сбора данных (3-5 мин)
```
AI: Расскажите о вашей компании — название и сайт?
User: Ромашка, https://romashka.ru
AI: Опишите ваш бизнес в 2-3 предложениях
User: Продаём цветы оптом и в розницу, Москва
AI: Кто ваши основные конкуренты? (необязательно)
User: Цветочный рай, Флорист.ру
AI: На какой email отправить отчёт?
User: ivan@romashka.ru
AI: Запускаю аудит для Ромашка по 6 платформам...
    ✅ Аудит запущен! [Смотреть прогресс →]
```

### 3. Страница прогресса (0-60 мин)
- Визуальный stepper с 7 шагами
- Обновляется каждые 5 секунд
- При готовности — авторедирект на отчёт

### 4. Email уведомление
Приходит при DELIVERING:
- Score компании (большая цифра с эмодзи)
- Ссылка на интерактивный отчёт
- Ссылка на PDF
- Краткий preview: топ-3 слабых места

### 5. Интерактивный отчёт
6 табов:
1. **Обзор** — score + топ проблемы
2. **Платформы** — RadialBar по каждой платформе
3. **Конкуренты** — таблица с сортировкой
4. **Слабые места** — аккордеон с severity
5. **План роста** — 3 колонки 30/60/90d
6. **Все запросы** — поиск/фильтр по 50 запросам

### 6. Post-Audit Chat (Standard — 10 вопросов)
```
[Кнопка: Спросить AI] — внизу экрана отчёта

User: Почему у нас score 28?
AI: Ваш score 28/100 складывается из трёх факторов:
    1. ChatGPT упоминает вас только в 12% ответов...
    2. Конкурент «Цветочный рай» встречается в 67%...
    Рекомендую начать с п.1 из вашего 30-дневного плана...
```
