# Product Vision

tags: #product #vision #design

## Что мы строим

Не просто чат-бот. **SaaS-инструмент для анализа AI-видимости бренда** с персональным кабинетом, историей аудитов, умным агентом и инструментами роста.

## Пользовательский сценарий (целевой)

```
1. Клиент приходит с лендинга
2. Вводит URL сайта → агент анализирует за 30 сек
3. Видит "Score preview" (Freemium) — 1 платформа
4. Выбирает тариф → оплачивает → полный аудит
5. Получает отчёт → задаёт вопросы в чате
6. (Advanced) Получает AI Website Fix через Stitch
7. Возвращается через месяц на re-audit
```

### Ключевые принципы UI
- Сайдбар с аудитами по нишам (как история ChatGPT)
- URL как точка входа, не чат
- Выбор тарифа встроен в диалог агента
- Кнопка оплаты — финальный шаг
- Тёмная тема, минимализм

## Тарифная сетка

| | Freemium | Basic $50 | Standard $150 | Advanced $300+ |
|---|---|---|---|---|
| Анализ сайта | ⚡ быстрый | ⚡ быстрый | 🔍 глубокий | 🔍 + конкуренты |
| Запросов | 5 (превью) | 15 | 50 | 150+ |
| Платформ | 1 (ChatGPT) | 3 | 6 | 8 (все) |
| Отчёт | Score only | HTML | PDF 28 стр. | PDF + Slides |
| Конкуренты | ❌ | ❌ | ✅ топ-5 | ✅ топ-10 |
| Action Plan | ❌ | ❌ | ✅ | ✅ + контент |
| Post-Audit Chat | ❌ | ❌ | 10 сообщ. | ♾️ |
| Google Stitch Fix | ❌ | ❌ | ❌ | ✅ |
| Content Agent | ❌ | ❌ | ❌ | ✅ |
| Скорость | 48ч | 48ч | 24ч | <12ч |

## Умный калькулятор тарифа

Агент определяет тариф автоматически после анализа сайта:
- Малый бизнес (1 локация) → Basic
- Средний (несколько услуг, конкуренты) → Standard  
- Крупный / агентство / мониторинг → Advanced

## Счётчик токенов (Admin)

Страница `/admin/usage`:
- Расход по каждой AI-платформе за текущий месяц
- % использования лимита
- Алёрты при достижении 80%
- Стоимость в $ за месяц
