## Общее впечатление

Полноценная платформа непрерывного GEO-мониторинга с богатым набором функций. Ориентирована на ongoing-слежку за брендом в AI-моделях, а не на разовый аудит. Интерфейс зрелый, онбординг продуманный. По функциональному объёму — самый сильный конкурент на рынке РФ.

## Онбординг (3 шага)

### Шаг 1: Создание бренда
- **AI-анализ сайта** (рекомендуемый путь): вводишь URL, система парсит сайт и автоматически заполняет:
  - Индустрию (напр. "Разработка и издательство видеоигр")
  - Описание бренда (несколько предложений)
  - Альтернативные названия бренда (транслитерация, сокращения, варианты на разных языках)
  - **Протестировано**: для astrum-entertainment.ru нашёл "Аструм Энтертейнмент", "Astrum", "Аструм", "Astrum Ent", "Astrum Entertainment Russia"
- **Ручное заполнение**: поля — название*, индустрия*, описание, альтернативные названия
- Альтернативные названия — ключевая фича: система отслеживает все варианты написания бренда в ответах AI

### Шаг 2: Конкуренты
- Добавление вручную: название + сайт (опционально)
- Возможность пропустить

### Шаг 3: Готово → дашборд

**Вывод по онбордингу**: AI-парсинг сайта при создании бренда — сильная UX-фича, снижает барьер входа. У KaliGEO аналога нет.

## Архитектура платформы

### МОНИТОРИНГ

#### Обзор (главный дашборд)
Метрики в шапке:
- **Видимость** — % промптов где бренд упомянут
- **Доля голоса (SoV)** — share of voice vs конкуренты
- **Упоминания** — абсолютное число за 30 дней
- **Тональность** — балл 0–100
- **Охват AI** — сколько из 9 платформ покрыто
- **Позиция в рейтинге** — место среди отслеживаемых брендов

Виджеты:
- **Динамика** (переключатели: Видимость / Позиция / Тональность / Видимость сайта)
- **Матрица бренд × платформа** — таблица с цветовой кодировкой по 9 платформам
- **Лидерборд** — бренды по убыванию Visibility / SoV / Position
- **Недавние упоминания** — последние тестовые прогоны
- **Лидерборд по темам** — ранг бренда в каждом тематическом кластере
- Фильтры: период (7д / 30д / 90д / свой), темы, экспорт

#### Промпты
- Управление списком промптов (по умолчанию генерируются автоматически)
- Ручной запуск теста
- Режим: ручной (N промп. × M мод. = запросов)
- **Подбор запросов** (новое): поиск релевантных запросов через sitemap + AI-кластеризацию
- Кнопки: Импорт / Статистика / Диалоги / AI-генерация / Создать
- Темы для группировки промптов
- Фильтры по моделям, тематике, статусу

#### Тональность
- Позитив / негатив / нейтральность по каждому промпту
- Динамика тональности во времени

#### Достоверность AI (только платный)
- **Факт-чек**: сверка утверждений AI с базой знаний бренда
- **Галлюцинации**: маркировка противоречий и неподтверждённых тезисов
- **Миф-сканер**: отслеживание устойчивых заблуждений в ответах AI о бренде

### АНАЛИТИКА

#### Конкуренты
- Сравнение видимости бренда vs конкурентов по каждой AI-модели
- Фильтры: период, модели
- Кнопка добавления конкурентов

#### Источники
- Какие сайты AI цитирует в ответах о нише
- 4 вкладки: **Источники бренда / Источники конкурентов / Сравнение / По промптам**
- Поиск по домену/URL
- Фильтр по периоду (30 дней по умолчанию)

#### Персоны (платный)
- AI-профили пользователей, которые ищут продукт/нишу
- Автоматическая кластеризация аудитории

#### Боты на сайте (платный)
- Трекинг визитов AI-краулеров через access-логи сайта
- Какие AI-боты индексируют сайт и когда

#### Эксперименты (платный)
- A/B-тесты: фиксация версии контента до и после правок
- Замер видимости бренда до/после изменений
- Агрегация экспериментов по бренду

#### Проверка домена (бесплатно)
- AI-готовность сайта: Schema.org, robots.txt, llms.txt, SSR
- История проверок
- «Новая проверка» по запросу

### СОЗДАНИЕ КОНТЕНТА

#### Рекомендации (платный)
- AI-брифы с планом действий: что написать, где опубликовать, какие запросы закрыть

#### Студия контента (платный)
- Генерация и оптимизация контента под AI-видимость
- Тестирование через RAG

#### Отчёт по бренду (платный, бета)
- Готовый PDF-отчёт со всеми метриками

#### Презентации (платный, alpha)
- Конструктор презентаций из виджетов — PDF за минуту

#### GEO Агент (бесплатно)
- Чат-помощник по данным аккаунта
- Задаёшь вопрос — получаешь ответ с графиками

#### Лаборатория контента (платный)
- Подробнее не доступно в триале

#### Карта бренда (платный)
- Граф связей: бренд ↔ AI-модели ↔ кластеры тем ↔ конкуренты ↔ источники
- Drill-down на каждом узле с детальными метриками
- Слепые зоны — промпты без покрытия

- RAG-индексируемая база фактов

- Используется в Достоверности AI и Лаборатории контента
- Источник правды для всех генеративных функций

#### Бренды
- Управление брендами (в триале лимит = 1 бренд)

## Инсайты для продукта KaliGEO

### 1. Мониторинг как тариф — критически важно
VisioBrand фокусируется именно на этом. Клиент хочет знать «что происходит сейчас», а не только «что было месяц назад». Наш разовый аудит — хорошее начало воронки, но retention строится на мониторинге.

### 2. Альтернативные названия бренда
VisioBrand автоматически ищет бренд по всем вариантам написания (транслитерация, сокращения). Это влияет на точность метрик. Нам нужно добавить это поле в форму заявки и учитывать при построении промптов.

### 3. Источники цитирования — незакрытая потребность у нас
Клиент хочет знать: «Какие сайты надо обработать, чтобы ChatGPT начал нас рекомендовать?» — это про источники. У нас есть source quality scoring, но нет явного UI для «каталога источников».

### 4. Достоверность AI / галлюцинации — уникальная ниша
AI-модели иногда врут о брендах (называют несуществующие продукты, путают факты). VisioBrand строит целый раздел на этом. Это реальная боль крупных брендов — и незакрытая у нас.

### 5. Проверка домена (llms.txt, robots.txt, SSR)
Быстрая техническая проверка AI-готовности сайта — хороший лид-магнит и point-of-entry. Бесплатный инструмент → конверсия в аудит.

### 6. Поиск запросов через sitemap + AI-кластеризация
Функция «Подбор запросов» — парсит sitemap и кластеризует темы с помощью AI. Полезно для клиентов, которые не знают какие промпты вводить. У нас GPT-4o-mini генерирует запросы автоматически, но без опоры на реальный контент сайта.

## Вывод

VisioBrand — сильнейший конкурент на рынке по объёму функций и зрелости продукта. Их стратегия: захватить monitoring-сегмент и добавлять контентные инструменты поверх. Слабое место: **нет глубины аудита и причинно-следственного анализа** — они показывают цифры, но не объясняют почему и что конкретно делать.

KaliGEO должен оставаться «самым умным аудитом» и добавить **лёгкий мониторинг** как retention-механику. Комбинация «глубокий старт + постоянный пульс» — незанятое место.
