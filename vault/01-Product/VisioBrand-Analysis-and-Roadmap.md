# KaliGEO vs VisioBrand: Анализ и Roadmap развития

> Дата анализа: 2026-05-27  
> Источник: visiobrand.io/ru, visiobrand.io/ru/offer

---

## Кто такой VisioBrand

**VisioBrand** — ближайший российский конкурент. Мониторинг видимости бренда в 9 AI-платформах (ChatGPT, Gemini, DeepSeek, Perplexity, Яндекс Алиса и др.). Ключевое позиционирование:

> "Упоминания — симптом. Роль — диагноз."

**Бизнес-модель:** ежемесячная подписка (не разовые аудиты), биллинг по запросам (1 запрос = 1 промпт × 1 AI-модель), 5-дневный триал.

## Roadmap улучшений

### 🔴 Sprint 1 — Фундамент (2-3 недели)

#### 1. Юридические документы
Создать страницы `/legal/offer`, `/legal/privacy`, `/legal/terms`, `/legal/refund`

Что включить в оферту:
- Предмет договора (мониторинг AI-видимости)
- Порядок оказания услуг
- 100% предоплата
- Автопродление подписки (уведомление за 24ч)
- Refund policy: 14 дней при неиспользовании
- Персональные данные (152-ФЗ для РФ)

**Файлы:** `app/legal/[slug]/page.tsx`, `content/legal/*.mdx`

#### 2. Magic-link авторизация
Вход без пароля — ссылка на email.

**Prisma:** добавить `UserSession { email, token, expiresAt }`  
**Routes:** `POST /api/auth/magic-link`, `GET /api/auth/verify`  
**Middleware:** защита `/dashboard/*`

### 🔴 Sprint 2 — Личный кабинет (3-4 недели)

#### Структура `/dashboard`

```
/dashboard                → Обзор: score, тренд, роль бренда
/dashboard/history        → Все аудиты, timeline, сравнение
/dashboard/competitors    → Конкурентная матрица в динамике
/dashboard/platforms      → Разбивка по каждой AI-платформе
/dashboard/queries        → Все запросы, фильтр
/dashboard/action-plan    → Текущий план действий
/dashboard/settings       → Частота, платформы, конкуренты, уведомления
/dashboard/billing        → Тариф, дата следующего списания, история
```

#### Ключевые виджеты на главной

- **Score ring** — общая видимость сейчас vs прошлый период
- **Trend chart** — динамика 30/90 дней (Recharts — уже есть в компонентах)
- **Роль бренда** — Главный выбор / Альтернатива / Упоминание (новая метрика)
- **Топ-3 слабых места** — кликабельный список с детализацией
- **Алерт-бейдж** — изменения с прошлого аудита

#### Классификация роли бренда

```typescript
// Добавить в QueryResult
brandRole: 'PRIMARY' | 'ALTERNATIVE' | 'MENTION' | null
```

**Логика:**
- `PRIMARY` — бренд упомянут первым / единственный ответ
- `ALTERNATIVE` — в списке вариантов, не первый
- `MENTION` — случайное упоминание без рекомендации

**Файлы:** `lib/analysis/mention-extraction.ts`, `prisma/schema.prisma`

### 🟠 Sprint 3 — Привлечение (2-3 недели)

#### Триальный доступ

**Рекомендуемая схема:**
1. Кнопка "Попробовать бесплатно" → ввод email
2. Magic-link → создание trial-аккаунта
3. 1 полный аудит BASIC-уровня бесплатно
4. Результаты в личном кабинете (побуждают купить дальше)
5. После 7 дней или использования → CTA на покупку

**Отличие от текущего freemium:** trial даёт полный аудит (15 запросов, 3 платформы) vs скан за 30 сек

#### HTML-скачивание и презентация

- **HTML-отчёт:** кнопка "Скачать HTML" → архив self-contained (CSS inline)
- **Презентация (.pptx):** `pptxgenjs` → 6-8 слайдов

Слайды презентации:
1. Обложка: бренд + дата + KaliGEO
2. Executive Summary: score, ключевой вывод
3. Конкурентная матрица
4. Топ-3 слабых места
5. Platform breakdown
6. Action Plan (кратко)
7. Следующие шаги
