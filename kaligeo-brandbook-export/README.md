# KaliGEO — Brand System Export

Этот пакет — исходник брендбука KaliGEO + правила, по которым ты делаешь
презентации, коммерческие предложения, лендинги и аудит-отчёты в этом стиле.

## Содержимое

- `Brandbook.html` — полный брендбук одной страницей. Открой в браузере, чтобы
  увидеть систему: логотип, цвета, типографику, компоненты, 9 эталонных слайдов.
  Это **источник истины**. Любые новые материалы должны выглядеть как продолжение этого документа.
- `STYLE_GUIDE.md` — конденсат: токены, типошкала, правила композиции, паттерны
  слайдов, copy voice, чек-лист «что нельзя». Читать перед стартом любого
  нового материала.
- `tokens.css` — готовые CSS-переменные и базовые классы. Импортируй в любой
  новый HTML-файл и работай уже на готовой основе.
- `slide-template.html` — минимальный шаблон одного слайда 1920×1080 со
  встроенной сеткой брендбука. Дублируй для новых слайдов/презентаций.

## Как этим пользоваться

### Для новой презентации / КП

1. Открой `Brandbook.html` и `STYLE_GUIDE.md`. Выбери 2–3 паттерна слайдов
   из брендбука, которые нужны (Title, Section header, Stat — sparkline,
   Stat — donut, Quote, Comparison, CTA).
2. Скопируй `slide-template.html` под рабочий файл.
3. Подключай `tokens.css`, используй уже готовые классы и переменные. Не вводи
   новые цвета, шрифты, радиусы — всё уже есть.
4. Один слайд = один экран 1920×1080. Используй starter `deck_stage.js` если
   делаешь deck — он уже включает скейл и навигацию.

### Для лендинга

1. Лендинг наследует ту же типошкалу и палитру.
2. Hero — паттерн `Title slide` (тёмный фон, серифный заголовок, lime accent
   на одном CTA, моно-метки).
3. Цифры/статы — паттерны `Stat — sparkline`, `Stat — donut`, `Comparison`.
4. Цитата клиента — паттерн `Quote slide`.
5. Финальная секция — паттерн `CTA slide`.

### Для аудит-отчёта

1. Cover = Title slide (имя клиента в монотеге вместо «Бутик-юрфирма»).
2. Section header перед каждым разделом аудита (1 страница, крупный серифный
   заголовок + одна строка курсивом).
3. Каждая метрика — отдельный слайд по паттерну `Stat —`.
4. Финал = CTA slide со следующим шагом, не «спасибо».

## Главные правила (одной строкой)

- **Серифные заголовки** (Instrument Serif), **гротеск для цифр и UI**
  (Inter Tight), **моно для меток** (JetBrains Mono).
- **Один акцент** — `#C8F24A` (lime). Никаких градиентов, теней, радужных
  палитр.
- **Контент по центру слайда по высеоте**, eyebrow сверху, type-tag снизу справа.
- **Один слайд = одна мысль.** Если на слайде 2 идеи — это 2 слайда.
- **Цифры крупные.** Главная метрика 96–144px. Не мельчи.
- **Никакой AI-слоповости** — emoji, иконок-стоков, абстрактных gradient blobs,
  generic SaaS-иллюстраций. См. STYLE_GUIDE → «Запрещённое».

Подробности в `STYLE_GUIDE.md`.
